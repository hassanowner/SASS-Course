
## What is While Loop in SCSS?

While Loop is a control structure in SCSS that repeats a block of code as long as a specified condition is true. It gives you control over the loop execution by letting you customize the condition and increment.

#### How it works:

· You set a starting variable value

· You write a condition that must be true for the loop to continue

· The loop runs until the condition becomes false

· You must change the variable inside the loop to avoid infinite loops

#### Simple Example:

```scss
$number: 1;

@while $number <= 3 {
  .box-#{$number} {
    width: 50px * $number;
  }
  $number: $number + 1;
}
```

Result in CSS:

```css
.box-1 { width: 50px; }
.box-2 { width: 100px; }
.box-3 { width: 150px; }
```

#### Explanation: The loop starts with $number = 1. It runs while $number is less than or equal to 3. Each time it creates a box class with increasing width, then adds 1 to $number.

## While Loop Examples

### Example 1: Simple - Font Size Classes

```scss
// Create font sizes from 12px to 24px
$size: 12;

@while $size <= 24 {
  .fs-#{$size} {
    font-size: #{$size}px;
    
    @if $size >= 18 {
      font-weight: bold;
    }
  }
  $size: $size + 2;
}
```

Result in CSS:

```css
.fs-12 { font-size: 12px; }
.fs-14 { font-size: 14px; }
.fs-16 { font-size: 16px; }
.fs-18 { font-size: 18px; font-weight: bold; }
.fs-20 { font-size: 20px; font-weight: bold; }
.fs-22 { font-size: 22px; font-weight: bold; }
.fs-24 { font-size: 24px; font-weight: bold; }
```

#### Explanation: This creates font size classes from 12px to 24px in 2px steps. Font sizes 18px and above become bold.

### Example 2: Medium - Progress Bar Steps

```scss
// Create progress bar widths from 0% to 100%
$percentage: 0;

@while $percentage <= 100 {
  .progress-#{$percentage} {
    width: $percentage * 1%;
    height: 20px;
    background-color: #4CAF50;
    border-radius: 10px;
    
    @if $percentage < 30 {
      background-color: #ff4444;  // Red for low progress
    } @else if $percentage < 70 {
      background-color: #ffaa44;  // Orange for medium progress
    } @else {
      background-color: #44ff44;  // Green for high progress
    }
  }
  $percentage: $percentage + 10;
}
```

Result in CSS:


#### Explanation: This creates progress bar classes from 0% to 100% in 10% steps. The color changes based on progress level:

· 0-20%: Red (low progress)

### Example 3: Advanced - Button Size Generator with Multiple Properties

```scss
// Create button sizes with multiple style variations
$size: 1;
$max-size: 5;

@while $size <= $max-size {
  .btn-size-#{$size} {
    // Base padding calculation
    $base-padding: 8px;
    $padding: $base-padding * $size;
    
    padding: $padding ($padding * 2);
    font-size: 12px + ($size * 2px);
    border-radius: 4px * $size;
    border: none;
    cursor: pointer;
    
    // Different colors for different sizes
    @if $size <= 2 {
      background-color: #3498db;  // Blue for small buttons
      color: white;
      
      &:hover {
        background-color: #2980b9;
      }
    } @else if $size <= 4 {
      background-color: #e74c3c;  // Red for medium buttons
      color: white;
      
      &:hover {
        background-color: #c0392b;
      }
    } @else {
      background-color: #2ecc71;  // Green for large buttons
      color: white;
      text-transform: uppercase;
      font-weight: bold;
      
      &:hover {
        background-color: #27ae60;
        transform: scale(1.05);
      }
    }
    
    // Add icons for specific sizes
    @if $size == 3 {
      &::before {
        content: "⭐";
        margin-right: 8px;
      }
    }
    
    @if $size == 5 {
      &::after {
        content: "✨";
        margin-left: 8px;
      }
    }
  }
  $size: $size + 1;
}
```

## Important Notes About While Loop

#### 1. Always Update the Variable

```scss
$counter: 1;

// Correct way - variable updates
@while $counter <= 5 {
  .box-#{$counter} { width: 50px * $counter; }
  $counter: $counter + 1; // Must update!
}
```

#### 2. Using Different Increments

```scss
$value: 5;

@while $value <= 30 {
  .step-#{$value} {
    margin: #{$value}px;
  }
  $value: $value + 5; // Increment by 5
}
// Creates: 5, 10, 15, 20, 25, 30
```

#### 3. Decreasing While Loop

```scss
$size: 100;

@while $size >= 20 {
  .box-#{$size} {
    width: #{$size}px;
    height: #{$size}px;
  }
  $size: $size - 10; // Decrease by 10
}
// Creates: 100, 90, 80, 70, 60, 50, 40, 30, 20
```

## While Loop vs For Loop

Feature For Loop While Loop

Syntax @for $i from 1 through 10 @while condition

Best for Fixed ranges Custom conditions

Increment Fixed (+1) Any increment you want

Control Less control More control

While loop is useful when you need custom increment values or complex stopping conditions!
